# A-share Investment Agent System

An AI-powered investment analysis system for A-share stocks, using multiple agents working together to provide investment decision support.

## Project Structure

```
src/                 # Source code
├── agents/          # Agent modules
├── data/            # Data fetching and processing
├── core/            # Core engine
├── models/          # Data models
├── utils/           # Utility functions
└── tools/           # Tool classes
```

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run specific modules as needed
python main.py
